# `<MagicTabSelect />`

### Overview

- This component is used for animating the tab selection indicator.

An example of this component is seen on the [Tabs example](/examples/tabs).

### Props

```ts
interface MagicTabSelectProps {
  id: string;
  children: JSX.Element;
  transition?: Transition;
}
```

| Prop name                                                 | Default value                                                                  | Description                                   |
| :-------------------------------------------------------- | :----------------------------------------------------------------------------- | :-------------------------------------------- |
| `id`                                                      | undefined                                                                      | Used to identify the component being animated |
| [`transition`](https://www.framer.com/motion/transition/) | [see here](https://gist.github.com/Etesam913/c56419e539573aece75bb7b9c6feef55) | The transition animation for the tab select.  |
